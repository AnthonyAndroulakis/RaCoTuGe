# RaCoTuGe
Random Constrained Tune Generator      

# prerequisites:
Mac OS      
python 3      
python modules:      
   audiolazy      
   midiutil      
   numpy      

# running RaCoTuge:
cd into RaCoTuGe directory      
```
chmod +x ./racotuge.sh      
```
```
./racotuge.sh 100 #for 100 tunes      
```
