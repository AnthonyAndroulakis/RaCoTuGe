# RaCoTuGe
Random & Constrained Tune Generator      

# prerequisites:
Mac OS      
python 3      
python modules:      
+ midiutil      
+ audiolazy      
+ numpy      

# running RaCoTuGe:
cd into RaCoTuGe directory      
```
chmod +x ./racotuge.sh      
```
```
./racotuge.sh 100 #for 100 tunes      
```
